Detailed Description

The World represents the whole program and manages creation of Thorin nodes (Defs).

Structural Defs are hashed into an internal HashSet. The getters just calculate a hash and lookup the Def, if it is already present, or create a new one otherwise. This corresponds to value numbering.

You can create several worlds. All worlds are completely independent from each other.

Note that types are also just Defs and will be hashed as well.
