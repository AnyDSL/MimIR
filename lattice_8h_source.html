25/// Specific [Bound](https://en.wikipedia.org/wiki/Join_and_meet) depending on @p Up.

28/// * @p Up = `true`: [Join](@ref thorin::Join) (aka least Upper bound/supremum/union)

93/// `test value, probe, match, clash` tests whether Test::value currently holds **type** Test::probe.

95/// * Test::probe is a **type**!
96/// * This operation yields Test::match, if `tt`, and Test::clash otherwise.
97/// @invariant
98/// * Test::value must be of type Join.
99/// * Test::match must be of type `A -> B`.
100/// * Test::clash must be of type `[A, probe] -> C`.
101/// @remark This operation is usually known as `case` but named `Test` since `case` is a keyword in C++.
102class Test : public Def {
