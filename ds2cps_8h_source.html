8/// This pass converts a direct style functions (abbreviated as ds) into CPS.
9/// the main conversion is shown in this pseudocode:
10/// ```
11/// h:
12/// b = f a
13/// C[b]
14/// ```
15/// becomes
16/// ```
17/// h:
18/// f'(a,h_cont)
19///
20/// h_cont(b):
21/// C[b]
22/// ```
23/// with the following types:
24/// ```
25/// f : A -> B
26/// f': .Cn [A, ret: .Cn[B]]
27/// ```
28/// The idea is to create a cps function for each ds function
29/// invoke the cps function with a continuation that takes the
30/// computation result and uses it in the original context.
