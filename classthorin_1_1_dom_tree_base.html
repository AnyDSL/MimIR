Detailed Description

template<bool forward>
class thorin::DomTreeBase< forward >

A Dominance Tree. The template parameter forward determines whether a regular dominance tree (true) or a post-dominance tree (false) should be constructed. This template parameter is associated with CFG's forward parameter.
