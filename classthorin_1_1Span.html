Detailed Description

template<class T>
class thorin::Span< T >

A container-like wrapper for an array.

The array may either stem from a C array, a std::vector, a std::initializer_list, an Array or another Span. Span does not own the data and, thus, does not destroy any data. Likewise, you must be carefull to not destroy data a Span is pointing to. Note that you can often construct a Span inline with an initializer_list: foo(arg1, {elem1, elem2, elem3}, arg3).
