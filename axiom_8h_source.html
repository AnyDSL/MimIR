58 /// Mangles @p s into a dense 48-bit representation.
59 /// The layout is as follows:
60 /// ```
61 /// |---7--||---6--||---5--||---4--||---3--||---2--||---1--||---0--|
62 /// 7654321076543210765432107654321076543210765432107654321076543210
63 /// Char67Char66Char65Char64Char63Char62Char61Char60|---reserved---|
64 /// ```
65 /// The `reserved` part is used for the Axiom::tag and the Axiom::sub.
66 /// Each `Char6x` is 6-bit wide and hence a dialect name has at most Axiom::Max_Dialect_Size = 8 chars.
67 /// It uses this encoding:
68 /// | `Char6` | ASCII |
