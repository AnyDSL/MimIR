Detailed Description

template<class Key, class T, class H = typename Key::Hash, hash_t StackCapacity = 4>
class thorin::HashMap< Key, T, H, StackCapacity >

This container is for the most part compatible with std::unordered_map. We use our own implementation in order to have a consistent and deterministic behavior across different platforms.
