Detailed Description

A thorin dialect.

This is used to load and manage a thorin dialect.

A plugin implementor should implement thorin_get_dialect_info and DialectInfo.

Dialect thorin::Dialect::load ( const std::string &  name,
Span< std::string >  search_paths 
)
static

Finds and loads a shared object file that implements the name thorin dialect.

If name is an absolute path to a .so/.dll file, this is used. Otherwise, "name", "libthorin_name.so" (Linux, Mac), "thorin_name.dll" (Win) are searched for in the search paths:

  1. search_paths, 2. env var THORIN_DIALECT_PATH, 3. "/path/to/executable"
