Detailed Description

template<bool forward>
class thorin::LoopTree< forward >

Calculates a loop nesting forest rooted at root_. The implementation uses Steensgard's algorithm. Check out G. Ramalingam, "On Loops, Dominators, and Dominance Frontiers", 1999, for more information.
